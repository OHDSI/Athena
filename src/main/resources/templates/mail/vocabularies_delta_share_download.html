Please download and load the Standardized Vocabularies as following:

  1. Click on this link to download the zip file. Typical file sizes, depending on the number of vocabularies selected, are between 30 and 400 MB.
  2. Unpack the downloaded zip file.
  3. Apply changes:
    1. For relative databases, execute the delta.sql script.
    2. For other databases, gather information about changes from the CSV files (refer to the readme for more details).

If you hit problems please use the OHDSI Forum pages, and somebody will help you. You will need to register.
